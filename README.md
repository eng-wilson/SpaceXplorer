## ðŸ’» Getting started

### Requirements


**Clone the project and access the folder**

```bash
$ git clone https://github.com/eng-wilson/SpaceXplorer.git && cd spacexplorer
```

**Follow the steps below**

```bash
# Install the dependencies
$ yarn

# Install Pods
$ npx pod-install ios

# Run Expo
$ expo start
```
